# 拡張モジュール適用報告書：柏市決算分析 (FY2024)

提供された「令和6年度 柏市歳入歳出決算書」に対し、**予算ポートフォリオ分析 (Budget Portfolio Analysis)** を適用しました。
この分析は、単なる金額の多寡ではなく、**「予算の巨大さ（金）」対「普及の広さ（人）」の歪み**を検出し、監査の優先順位を付けることを目的としています。

---

## 1. 基礎定数の設定 (Kashiwa Constants)

柏市は中核市（人口約43.5万人）であるため、国全体の「標準ブロック（人口7.2万人）」に対して、**約6.0倍のスケール**を持ちます。これを考慮し、分析係数を補正します。

*   **P_kashiwa (柏市人口):** 435,000人
*   **N_block (柏市ブロック係数):** `435,000 ÷ 72,000 ≒ 6.0`
    *   *意味:* 柏市の施策は、標準的な自治体の6倍の規模（または予算）があって初めて「平均的」とみなされます。

---

## 2. 歪み検出ケーススタディ (Distortion Detection)

決算書から特徴的な事業を抽出し、**予算歪み指数 (D_index)** を算出しました。

### ケースA：ゼロカーボンシティ促進総合補助金 (環境部)
*   **参照:** 歳出 第4款 衛生費 / 第5項 環境対策費 (PDF p.388/420)
*   **決算額:** 29,850,000円 (約3,000万円)
*   **概要:** 太陽光発電設備や省エネ機器導入への補助金。
*   **推計利用者数:** 予算規模と相場（1件10万円補助と仮定）から、約300世帯と推計。

**【分析計算】**
1.  **予算インパクト (I_budget):**
    標準的な単一補助事業予算を1,000万円とした場合、柏市規模(x6)なら6,000万円が適正ライン。
    `I_budget = 3,000万 ÷ 6,000万 = 0.5`
    *評価: 予算規模自体は控えめ。*

2.  **普及インパクト (I_coverage):**
    `I_coverage = 300人 ÷ 72,000人 ≒ 0.0042`
    *評価: 誤差レベル。市民の0.1%にも届いていない。*

3.  **歪み指数 (D_index):**
    `D_index = 0.5 ÷ 0.0042 ≒ 119`

**【判定】 第4象限（歪み・バラマキ型）**
*   **結論:** **D = 119** は極めて高い数値です。特定の少人数（300世帯）に対し、税金が集中投下されています。
*   **監査ポイント:** 「一部の持ち家層への優遇ではないか？」「CO2削減量というKGIに対し、3,000万円のROIは正当か？」

---

### ケースB：柏駅周辺まちづくり事業 (土木部)
*   **参照:** 歳出 第8款 土木費 / 第4項 都市計画費 (PDF p.479/511)
*   **決算額:** 1,000,000,000円 (**10億円**)
*   **概要:** 北口・東口等の再整備、用地買収等を含む。
*   **利用者数:** 柏駅利用者（市民の多くが恩恵を受けるインフラ）。

**【分析計算】**
1.  **予算インパクト (I_budget):**
    大規模インフラの標準を5億円とした場合、柏市規模(x6)なら30億円が基準。
    `I_budget = 10億 ÷ 30億 ≒ 0.33`
    *評価: 意外にも、都市規模に対して投資額は抑制的である。*

2.  **普及インパクト (I_coverage):**
    駅利用者を市民の50%と仮定（21.7万人）。
    `I_coverage = 217,000人 ÷ 72,000人 ≒ 3.0`

3.  **歪み指数 (D_index):**
    `D_index = 0.33 ÷ 3.0 ≒ 0.11`

**【判定】 第2象限（優良インフラ・低コスト広範囲）**
*   **結論:** 10億円という絶対額は巨額ですが、柏市の人口規模と駅の公共性を鑑みると、**「1人あたりのコストは極めて低い」**高効率な投資です。
*   **監査ポイント:** 逆に「投資不足ではないか？」「老朽化対策は十分か？」という視点が必要。

---

### ケースC：不登校児童生徒の支援 (学校教育部)
*   **参照:** 歳出 第10款 教育費 / 第3項 教育指導費 (PDF p.534/566)
*   **決算額:** 169,459,639円 (約1.7億円)
*   **概要:** スクールソーシャルワーカー報酬、教育支援センター指導員報酬など。
*   **ターゲット:** 不登校児童生徒（柏市小中学生約3.5万人のうち、不登校傾向を3%と仮定 ≒ 1,000人）。

**【分析計算】**
1.  **予算インパクト (I_budget):**
    教育支援の標準ブロックを2,000万円とした場合、柏市基準は1.2億円。
    `I_budget = 1.7億 ÷ 1.2億 ≒ 1.42`
    *評価: 基準よりやや手厚い予算配分。*

2.  **普及インパクト (I_coverage):**
    `I_coverage = 1,000人 ÷ 72,000人 ≒ 0.014`

3.  **歪み指数 (D_index):**
    `D_index = 1.42 ÷ 0.014 ≒ 101`

**【判定】 第4象限（ケア型・高福祉）**
*   **結論:** **D = 101** と高い数値が出ますが、これは「歪み」というより**「セーフティネットのコスト」**と解釈すべき特性です。ただし、数値上は「高コスト・少数対象」であるため、事業の効率性（人件費の中抜きがないか等）は厳しくチェックする必要があります。
*   **監査ポイント:** 1.7億円のほとんどが「報酬（人件費）」です。「相談員が実際に何件対応したか（単価）」を算出する必要があります。

## 3. Pythonによる自動検証コード

以下のスクリプトを使用することで、決算書の任意の項目について、この「歪み判定」を即座に行うことができます。

```python
def calculate_distortion(budget_yen, beneficiaries, city_pop=435000):
    """
    柏市予算ポートフォリオ分析関数
    budget_yen: 事業決算額 (円)
    beneficiaries: 推定受益者数 (人)
    city_pop: 柏市人口 (デフォルト43.5万人)
    """
    
    # 定数定義
    NATIONAL_STD_BLOCK_POP = 72000  # 標準ブロック人口
    STD_MUNI_BUDGET_UNIT = 10000000 # 標準的な単一事業の予算単位(1000万円と仮定)
    
    # 柏市の規模係数 (Kashiwa Scale Factor)
    scale_factor = city_pop / NATIONAL_STD_BLOCK_POP
    
    # 1. 予算インパクト (I_budget)
    # 柏市の適正予算 = 標準単位 * 規模係数
    kashiwa_std_budget = STD_MUNI_BUDGET_UNIT * scale_factor
    i_budget = budget_yen / kashiwa_std_budget
    
    # 2. 普及インパクト (I_coverage)
    # 全国標準ブロックに対するカバー率
    i_coverage = beneficiaries / NATIONAL_STD_BLOCK_POP
    
    # 3. 歪み指数 (D_index)
    # 0除算回避
    if i_coverage == 0:
        return 0, 0, float('inf'), "データなし"
        
    d_index = i_budget / i_coverage
    
    # 判定ロジック
    verdict = ""
    if d_index < 1:
        verdict = "【第2象限】高効率・インフラ型 (Excellent)"
    elif 1 <= d_index < 10:
        verdict = "【第1/3象限】適正範囲 (Normal)"
    elif 10 <= d_index < 50:
        verdict = "【注意】高コスト・少数対象 (Warning)"
    else:
        verdict = "【第4象限】歪み・要監査 (Distorted)"
        
    return i_budget, i_coverage, d_index, verdict

# --- 実行例 ---
projects = [
    {"name": "A. ゼロカーボン補助金", "budget": 29850000, "users": 300},
    {"name": "B. 柏駅周辺整備", "budget": 1000000000, "users": 217000},
    {"name": "C. 不登校支援", "budget": 169459639, "users": 1000}
]

print(f"{'事業名':<20} | {'予算インパクト':<15} | {'普及インパクト':<15} | {'歪み指数':<10} | {'判定'}")
print("-" * 90)

for p in projects:
    ib, ic, d, v = calculate_distortion(p["budget"], p["users"])
    print(f"{p['name']:<20} | {ib:<15.2f} | {ic:<15.4f} | {d:<10.1f} | {v}")
```

## 監査チームへの提言

PDF内の**「扶助費（補助金）」**の項目（特に環境・福祉・商工振興）を重点的にこの式に当てはめてください。
$D_{index} > 100$ を超える事業は、**「効果検証が不十分なまま漫然と継続されているバラマキ」**である可能性が極めて高いです。
